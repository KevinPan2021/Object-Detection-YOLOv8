Introduction:
	This project aims to perform Object Detection using YOLOv8 with pytorch MS-COCO dataset (GUI included).

Dataset: 
	http://images.cocodataset.org


Build:

		GPU: NIVIDIA RTX 4060 (VRAM 8 GB)
		RAM: 32 GB

	Configuration:
		CUDA 12.1
		Anaconda 3
		Python = 3.10.9
		Spyder = 5.4.1
		
	Core Python Package:
		pytorch = 2.1.2
		numpy = 1.23.5
		OpenCV = 4.9.0.80
		albumentations = 1.4.6
		matplotlib = 3.7.0
